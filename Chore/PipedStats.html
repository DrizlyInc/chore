    class Chore::PipedStats

    Public Class Methods

    new(pipe_id,listener,bucket_size=nil) click to toggle source

    PipedStats is a neat trick to avoid having to let our Worker know whether or not it's running forked. After we fork a child, we reset Chore.stats to be an instance of PipedStats. This class overrides a couple of key methods to send data across the pipe, instead of directly into the in memory stats.

    # File lib/chore/strategies/worker/forked_worker_strategy.rb, line 36
def initialize(pipe_id,listener,bucket_size=nil)
  super(bucket_size)
  @pipe_id, @listener = pipe_id, listener
end
